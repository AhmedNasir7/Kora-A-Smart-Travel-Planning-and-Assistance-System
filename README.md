# Kora – Smart Travel Assistant

Travel planning is stressful because flight times, hotel check-ins, packing needs, and documents are scattered across different apps, emails, and notes. Kora solves this by providing a single platform that organizes everything and proactively reminds you what to do and when.

## What Kora does
- Manage trips (destination + dates)
- Track flight/bus/train schedules and hotel check-in/out times
- Create packing lists per trip
- Store important documents (tickets, IDs)
- Receive visual/audio reminders for upcoming tasks
- View a clear trip timeline
- Dashboard overview of upcoming trips and reminders

## What makes Kora different
- Focus on guidance, not just information display
- Combines packing, documents, schedules, and reminders
- Step-by-step trip assistance
- Designed for minimal user effort and clarity

## Tech Stack
- Frontend: Next.js
- Backend: NestJS (REST APIs)
- Database & Auth: Supabase
- Notifications: Firebase Cloud Messaging (FCM)
- Design: Figma + Canva

## Out of Scope (for this prototype)
- No flight/hotel booking
- No real-time ticket purchase
- No integration with airline/hotel systems
- No AI predictions using live external data
